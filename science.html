Our Research Foundations

Built on decades of scientific discovery in cognitive neuroscience and learning theory

Neuroplasticity

The brain's remarkable ability to reorganize itself by forming new neural connections throughout life.

Key Findings:

  • Brain structure changes with learning
  • New neurons can form in hippocampus
  • Neural pathways strengthen with practice
  • Age is no barrier to learning
Draganski et al., 2004 - Nature

Working Memory

The cognitive system responsible for temporarily holding and processing information.

Key Findings:

  • Capacity can be expanded with training
  • Crucial for complex reasoning
  • Linked to academic achievement
  • Trainable through specific exercises
Baddeley, 2000 - Trends in Cognitive Sciences

Neural Networks

Complex interconnected systems of neurons that process and transmit information.

Key Findings:

  • Distributed processing across brain regions
  • Efficiency improves with expertise
  • Default mode network activation
  • Connectivity predicts learning ability
Bassett & Mattar, 2017 - Network Neuroscience

Key Brain Regions We Target

Understanding the neural circuitry behind learning and memory

Hippocampus

Memory Formation & Spatial Navigation

Critical for forming new memories and spatial navigation. Shows remarkable neuroplasticity in response to learning.

Primary Functions:

  • Long-term memory formation
  • Spatial memory and navigation
  • Memory consolidation
  • Contextual learning
Training Impact: 25-40% increase in grey matter density with memory training

Prefrontal Cortex

Executive Functions & Working Memory

Orchestrates complex cognitive behaviors, decision making, and moderates social behavior.

Primary Functions:

  • Working memory
  • Decision making
  • Problem solving
  • Attention control
Training Impact: Enhanced connectivity and efficiency with cognitive training

Parietal Lobe

Mathematical Cognition & Sensory Integration

Processes sensory information and is crucial for mathematical thinking and spatial reasoning.

Primary Functions:

  • Mathematical processing
  • Sensory integration
  • Spatial reasoning
  • Attention shifting
Training Impact: Improved activation patterns with math training
